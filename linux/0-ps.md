`ps -ef|head -n 10` 标准输出
`ps aux|head -n 10`  BSD syntax
x 就是打印 BSD 风格，-f 同样也是打印格式的问题。
```
STAT              状态

          R：runing 运行态 

          S：可中断睡眠态 

          D：不可中断睡眠态

          T：停止态

          Z：僵尸态

          s：表示这个进程是个领导者进程

          +：该进程为前台进程

          l：该进程是多线程进程

         N：低优先级进程

         <:高优先级进程 

         []:表示这个一个内核线程
```
